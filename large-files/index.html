Large files.

xlsx-for-ai reads workbooks up to 10 MB for free, like everything else it does. Files past that need more memory than we provision for a single job — so right now they're declined rather than run slowly or unreliably.

Why there's a limit

Reading a workbook into memory, evaluating its formulas, and writing a reviewed copy all hold the file — and intermediate working copies of it — in RAM at once. For most workbooks that's comfortable. Past about 10 MB it stops being comfortable on the memory we run each job with, and we'd rather decline cleanly than give you a slow or flaky result.

We're weighing dedicated large-file support

We're considering running large files on dedicated machines with more memory, so the same review works on bigger workbooks. Before we build it, we want to know if it's actually wanted — and what it's worth to whoever needs it.

Everything up to 10 MB stays free. This is about whether a bigger-file option should exist alongside it — not a change to what's free today, and there's no price set.
