Let your agent inventory the tabs before processing

When an agent receives an xlsx file and needs to route data from the right tab to the right downstream system, it can call the same sheet-listing function directly — names, row counts, column headers, hidden tab detection — before any further processing. Wire xlsx-for-ai into your agent and tab discovery becomes one tool call, not a manual opening step.

The tools this page runs

Your agent calls xlsx_list_sheets (tab names, dimensions, visibility) and xlsx_describe (per-column count, nulls, unique values, and type) to map a workbook without dumping every cell.

  • xlsx_describe options: sheet, max_rows, header_row.
