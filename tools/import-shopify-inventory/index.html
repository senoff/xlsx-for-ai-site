What it does

Shopify's inventory import matches each row to a SKU at a location and sets its available quantity. Your source counts rarely arrive in that exact shape. This tool reads your file, maps your SKU, location, and quantity columns to the right inventory fields, and writes a clean import CSV you can review and upload in your Shopify admin.

It flags what it won't write

Inventory quantities are whole numbers. A decimal count, a blank quantity, or a row missing its SKU can't be safely written — so instead of guessing, we hold those rows back and list them under what needs you, with the reason. Everything we did map shows up under what we mapped. You see the full before-you-import picture in one place.

Reviewed file, not an automatic write

Because an inventory import overwrites your live counts with no automatic undo, this tool hands you a reviewed file to check and import yourself — it never writes to your store directly. Your uploaded file is read in memory and never stored.
