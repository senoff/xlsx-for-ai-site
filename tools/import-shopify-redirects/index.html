What it does

Shopify's URL redirects import expects a “from” path and a “to” path per row, each in a specific shape. Redirect lists gathered during a site move rarely arrive that way — headers vary, and some rows carry a full URL where a path belongs. This tool reads your file, maps your old-path and new-path columns to the right redirect fields, and writes a clean import CSV you can review and upload in your Shopify admin.

One review surface for the whole set

Every pair you gave us shows up under what we mapped, and any row we couldn't use — a blank target, or an absolute URL where Shopify wants a path — lands under what needs you, with the reason. You review the entire redirect set in one place before you import it, so a malformed row doesn't slip through unnoticed.

Reviewed file, not an automatic write

This hands you a reviewed import file to check and upload yourself — it never writes to your store directly. Your uploaded file is read in memory and never stored.
