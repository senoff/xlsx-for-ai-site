What it cleans

The cleaner runs a full sweep for the grime that piles up in exported spreadsheets: stray leading and trailing spaces, numbers stored as text, fully-duplicate rows, empty trailing rows, inconsistent “not available” markers, mixed date formats, and label spellings that should match but don’t. Each fix is applied only when it’s unambiguously safe.

The before→after ledger is the point

Nothing changes silently. Every fix shows up under what we did, and anything we deliberately left alone — blank cells we won’t invent data for, a column of repeated values that’s yours to keep or drop — shows up under what we didn’t touch, with the reason. You see the whole picture before you download.

Your original file is safe

The page reads your file in memory, cleans a copy, and double-checks that copy opens the same in two independent spreadsheet engines. Your uploaded .xlsx is never changed and never stored — you download a fresh, clean file, so nothing is ever lost.
