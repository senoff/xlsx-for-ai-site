Let your agent reconcile files as part of an automated workflow

When a reporting pipeline needs to verify that a newly received data file agrees with last period's file before processing it, it can run the same reconciliation automatically — rows changed, rows added, rows removed — and surface only the discrepancies that need human review. Wire xlsx-for-ai into your agent and reconciliation becomes a gate in the pipeline, not a manual step before it.

The tool this page runs

Your agent calls xlsx_diff with the two files and gets a deterministic, cell-level diff — changed, added, and removed values, plus a stable diff_hash for caching.

  • Option: sheet — limit the comparison to a single tab.
