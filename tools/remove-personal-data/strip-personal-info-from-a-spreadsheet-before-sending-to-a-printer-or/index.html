When an agent pipeline processes user-uploaded spreadsheets before passing them to an LLM, it can run the same column-level redaction automatically — no user upload step, no browser. The tool this page runs

Your agent calls xlsx_redact; by default it masks PII values while preserving formulas, comments, named ranges, and styles.

  • Options: strip_formulas, strip_comments, strip_macros, strip_metadata to remove those too; mode: scaffold to keep only the structure.
